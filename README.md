
# 📚 Paju Publishing Chatbot (파주 출판단지 안내 챗봇)

> 이미지로 장소를 인식하고, 행사 정보를 RAG로 검색해 안내하는 파주 출판단지 전용 AI 챗봇입니다.

---

## 🚀 프로젝트 개요

이 프로젝트는 **파주 출판단지 내 장소 인식 + 행사 안내 챗봇**입니다.

* 이미지를 업로드하면 ResNet18 모델이 장소를 예측
* 사용자가 텍스트로 질문하면 GPT + Chroma 기반 RAG 검색으로 행사 정보 제공

---



## 실행방법

### 행사 json 파일 불러오기 (한번만 실행)

```bash
python event_loader.py
```

실행 후 `paju_db/` 폴더가 생성되며, 모든 행사 데이터가 벡터DB로 저장됩니다.

---


### 챗봇 실행

```bash
python chat.py
```

이후 콘솔에서:

```
1. 텍스트 질문
2. 이미지 업로드
```

를 선택하여 대화 시작.

---

## 🧠 주요 기능

| 기능             | 설명                                        |
| -------------- | ----------------------------------------- |
| 🖼️ 이미지 인식     | ResNet18 모델이 장소 인식 (e.g. 지혜의숲, 프로방스 마을 등) |
| 💬 텍스트 질의응답    | Azure GPT-4o 모델로 자연스러운 대화                 |
| 🔍 행사 검색 (RAG) | ChromaDB + text-embedding-3-small 임베딩     |
| 🧾 안내문 자동 생성   | "행사명/장소/일시/요약" 정보를 포맷팅해 제공                |
