# Paju Publishing Chatbot (파주 출판단지 안내 챗봇)

이미지로 장소를 인식하고, 행사 정보를 RAG로 검색해 안내하는 파주 출판단지 전용 AI 챗봇
https://dacon.io/competitions/official/236612/overview/description
 장려상 수상

### 1. 주요 기능

1. 이미지 장소 인식
   - 사용자가 업로드한 이미지를 분석하여 파주 출판단지 내 장소를 인식
   - ResNet18 모델을 파주 출판단지 장소 데이터로 파인튜닝하여 사용
   - 11개 장소 분류 가능 (지혜의숲, 문학동네, 열화당, 김영사 사서점, 북카페눈, 은하수 출판사, 음악세계, 어린이집, 파인드아웃, milkbook, Sbg파주사옥)
   - 이미지 전처리: 224x224 리사이즈 후 정규화

2. 텍스트 질의응답
   - Azure OpenAI GPT 모델을 활용한 자연어 대화
   - 사용자별 대화 세션을 유지하여 맥락 기반 응답 제공
   - 파주 출판단지 관련 질문에 대해 2~3줄 요약 형식으로 답변

3. 행사 검색 (RAG)
   - ChromaDB 벡터 데이터베이스에 행사 정보 저장
   - Azure OpenAI Embeddings를 사용하여 행사 데이터 임베딩
   - LangChain 기반 RAG 파이프라인 구현
   - 사용자 질문과 유사한 행사 정보를 검색하여 GPT에 컨텍스트로 전달
   - 청킹 설정: chunk_size 800, chunk_overlap 100

4. 안내문 자동 생성
   - 검색된 행사 정보를 GPT가 보기 좋게 포맷팅
   - 제목, 일시, 장소, 주최, 요약, 신청방법 형식으로 정리하여 출력

5. 스탬프 적립
   - 이미지 업로드 시 해당 장소의 스탬프 적립 기능
   - 사용자별 적립된 스탬프 목록 관리


### 2. 실행 방법

1. 행사 json 파일 불러오기 (최초 1회 실행)

```bash
python event_loader.py
```

실행 후 paju_db/ 폴더가 생성되며, 모든 행사 데이터가 벡터DB로 저장됩니다.

2. 챗봇 실행

```bash
python chat.py
```

3. 실행 후 콘솔에서 텍스트 질문 또는 이미지 업로드를 선택하여 대화 시작
